Controls

You can pause/resume the broadcast using the pause button, this stops your watch editor from updating whatever the teacher is broadcasting.
The slider can be used to resize the editors.
You can also request code review from your teacher. Clicking this button notifies your teacher that you want a code review.

Attendee list

All class attendees are in this list, including your teacher. Click their name to view their code. You can hide this list by clicking the sidebar button.

Watch people code

This is a read-only editor that displays code that your teacher is broadcasting.

This is your workspace. You can follow along with your teacher or try something different.
